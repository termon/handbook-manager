# Laravel Blade Starter Kit

This starter kit gives you a Laravel 13 application with authentication, a shared Blade UI library, Livewire support, and example project structure already in place.

## Core Pages

- [FAQ](/help?page=faq)
- [Help System](/help?page=help-docs)
- [Traits](/help?page=features/traits)
- [Impersonation](/help?page=features/impersonation)

## Included

- Laravel 13
- `termon/ui`
- Livewire 4
- Alpine.js
- Tailwind CSS 4
- seeded users and role enum support
- user management with impersonation
- feature help pages under `features/`

## Use This Starter Kit For

- admin panels and internal tools
- server-rendered Laravel apps with Blade
- projects that need reusable UI components quickly
- applications that benefit from Livewire-based interactivity
