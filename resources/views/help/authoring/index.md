# Authoring Overview

This section is for authorised users who can create or edit handbook content in the admin area.

## Access

- Open **Handbooks** from the admin area.
- Select an existing handbook to edit it, or create a new handbook if your role allows it.
- Use the panel navigation to switch between the editor, preview, details, and images panels.

## Editing Pages

- Use **Create page** to add a new page to the current handbook.
- Use **Add shared** to attach a shareable page from another handbook when your permissions allow it.
- Edit the page title and markdown body in the **Markdown editor** panel.
- Save changes with **Save page**.

## Adding Images To A Page

- Open the **Images** panel for the current handbook.
- Upload an image with either the single-file or multi-file uploader.
- Find the uploaded image in the list and use **Copy markdown** to copy its markdown snippet.
- Return to the **Markdown editor** panel and paste that snippet into the page body where the image should appear.
- Save the page after inserting or updating image markdown.

## Previewing Content

- Open the **Page preview** panel to review the rendered version of the current page.
- Use this panel to check headings, links, markdown formatting, and embedded images before publishing.

## Unsaved Changes

- Save your page before moving to another page, switching to another handbook, or following another navigation path in the admin area.
- Navigating away from the current page editor discards unsaved page title and markdown changes.
- This also applies if you paste an image markdown snippet into the editor and then leave the page without saving.

## Managing Handbook Details

- Open the **Handbook details** panel to update the handbook title and description.
- Admins can assign ownership to an author.
- Listed handbooks can appear in the public handbook listing. Unlisted handbooks remain hidden from that index.

## Working With Images

- Open the **Images** panel to upload handbook images.
- Use **Single image upload** when you want to review an overwrite and optionally provide custom alt text.
- Use **Multiple image upload** when you want to upload several images in one batch.
- After upload, copy the markdown snippet for an image and paste it into the page body where you want it to appear.

## Good Practice

- Use clear page titles and image alt text.
- Preview before saving when you have changed layout-heavy markdown.
- Keep image filenames descriptive so the copied markdown path is easy to recognise later.
- Treat page navigation as a boundary: save first, then move elsewhere.
